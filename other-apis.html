other APIs

There are a lot of cool APIs out there! Here's just a super short (and in no way comprehensive) selection of some interesting APIs to try out:

  • GitHub Rest API: lets you get github user stats (cool to display on a personal website!)
  • NASA APIs: allow access to everything from weather on Mars to astrology images of the day
  • PokeAPI: gives access to massive amounts of pokemon data!
  • ...and many more!
